Slotti Ajanvaraus -lisäosa helpottaa ajanvarauspainikkeen tai upotetun varaussivun lisäämistä WordPress-sivuille.

== Kuvaus ==

Slotti tarjoaa verkkosivujanne täydentävän ajanvaraussivun, jonka avulla asiakkaanne voivat varata aikoja palveluunne. Tämä lisäosa helpottaa "Varaa nyt"-painikkeen tai upotettavan varaussivun lisäämistä sivuillenne.

Slotti Ajanvaraus -lisäosan tarjoamien shortcodejen avulla voitte
- Lisätä varauspainikkeen, joko valmiiksi tyyliteltynä tai omilla tyyleillänne
- Upottaa varaussivun omille sivuillenne
- Välittää tiedon kävijältä saadusta GDPR-seurantaluvasta analytiikkatoimintoja varten

Tarkemmat ohjeet käytöstä löytyvät [Slotin Tietopankista](https://slotti.fi/kb/fi/slotti-ajanvaraus-wordpress-sivuille-pluginin-kaytto/).

Lisäosan käyttöön tarvitaan slotti.fi -palvelu. Voit kysyä lisää osoitteesta info@slotti.fi.

== Asennus ==

1. Lataa slotti-wp.zip WordPressiin tai asenna plugin suoraan WordPressin "Lisäosat"-näkymästä valitsemalla "Lisää Uusi" ja etsimällä "Slotti Ajanvaraus"-lisäosa.
1. Ota plugin käyttöön
1. Katso pluginin ohjeet käyttöön [Slotin Tietopankista](https://slotti.fi/kb/fi/slotti-ajanvaraus-wordpress-sivuille-pluginin-kaytto/).
